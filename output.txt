
-----------------------------------------------------------------------------
------------------------------ SISTEMA DOMOTICO -----------------------------
-----------------------------------------------------------------------------
Le operazioni che puoi eseguire con il tuo sistema domotico sono le seguenti: 
  # set ${DEVICENAME} on
  # set ${DEVICENAME} off
  # set ${DEVICENAME} ${START} [${STOP}]
  # rm ${DEVICENAME}
  # show
  # show ${DEVICENAME}
  # set time ${TIME}
  # reset time
  # reset timers
  # reset all
  # add
  # erase ${DEVICENAME}
Per uscire digitare: exit

[00:00] set Lavastoviglie on
[00:00] Il dispositivo Lavastoviglie si e' acceso.

[00:00] set Lavatrice on
[00:00] Il dispositivo Lavatrice si e' acceso.

[00:00] set Televisore 00:30 1:00
[00:00] Il dispositivo Televisore e' un dispositivo a ciclo prefissato. Impossibile impostare un timer di spegnimento. Nessun timer impostato.

[00:00] set Televisore 00:30
[00:00] Impostato un timer per il dispositivo Televisore dalle ore [00:30] alle ore [01:30].

[00:00] set Televisore on
[00:00] Televisore non acceso. Limite di potenza raggiunto.

[00:00] set Pompa di calore + termostato 00:10
[00:00] Impostato un timer per il dispositivo Pompa di calore + termostato dalle ore [00:10].

[00:00] set time 00:25
[00:00] L'orario attuale e' 00:00.
[00:10] Pompa di calore + termostato non acceso. Limite di potenza raggiunto.
[00:25] L'orario attuale e' 00:25.

[00:25] set time 00:45
[00:25] L'orario attuale e' 00:25.
[00:30] Televisore non acceso. Limite di potenza raggiunto.
[00:45] L'orario attuale e' 00:45.

[00:45] set Lavatrice off
[00:45] Il dispositivo Lavatrice si e' spento.

[00:45] set Televisore on
[00:45] Il dispositivo Televisore si e' acceso.

[00:45] set time 3:00
[00:45] L'orario attuale e' 00:45.
[01:45] Il dispositivo Televisore si e' spento.
[03:00] L'orario attuale e' 03:00.

[03:00] exit
