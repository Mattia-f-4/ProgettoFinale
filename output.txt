
-----------------------------------------------------------------------------
------------------------------ SISTEMA DOMOTICO -----------------------------
-----------------------------------------------------------------------------
Le operazioni che puoi eseguire con il tuo sistema domotico sono le seguenti: 
  # set ${DEVICENAME} on
  # set ${DEVICENAME} off
  # set ${DEVICENAME} ${START} [${STOP}]
  # rm ${DEVICENAME}
  # show
  # show ${DEVICENAME}
  # set time ${TIME}
  # reset time
  # reset timers
  # reset all
  # add
  # erase ${DEVICENAME}
Per uscire digitare: exit

[00:00] set Lavatrice on
[00:00] Il dispositivo Lavatrice si e' acceso.

[00:00] set Asciugatrice 2:00
[00:00] Impostato un timer per il dispositivo Asciugatrice dalle ore [02:00] alle ore [03:00].

[00:00] set Impianto fotovoltaico 3:00 4:00
[00:00] Impostato un timer per il dispositivo Impianto fotovoltaico dalle ore [03:00] alle ore [04:00].

[00:00] set time 5:00
[01:50] Il dispositivo Lavatrice si e' spento.
[02:00] Il dispositivo Asciugatrice si e' acceso.
[03:00] Il dispositivo Asciugatrice si e' spento.
[03:00] Impianto fotovoltaico non acceso. Limite di potenza raggiunto.
[04:00] Impianto fotovoltaico gia' spento. QUESTO NON DOVREBBE ACCADERE, CONTROLLARE CHE PROBLEMI CI SONO
[05:00] Orario impostato.

[05:00] exit
