
-----------------------------------------------------------------------------
------------------------------ SISTEMA DOMOTICO -----------------------------
-----------------------------------------------------------------------------
Le operazioni che puoi eseguire con il tuo sistema domotico sono le seguenti: 
  # set ${DEVICENAME} on
  # set ${DEVICENAME} off
  # set ${DEVICENAME} ${START} [${STOP}]
  # rm ${DEVICENAME}
  # show
  # show ${DEVICENAME}
  # set time ${TIME}
  # reset time
  # reset timers
  # reset all
  # add
  # erase ${DEVICENAME}
Per uscire digitare: exit

[00:00] set Impianto fotovoltaico 3:00 4:00
[00:00] Impostato un timer per il dispositivo Impianto fotovoltaico dalle ore [03:00] alle ore [04:00].

[00:00] set time 5:00
[03:00] Il dispositivo Impianto fotovoltaico si e' acceso.
[04:00] Il dispositivo Impianto fotovoltaico si e' spento.
[05:00] Orario impostato.

[05:00] exit
